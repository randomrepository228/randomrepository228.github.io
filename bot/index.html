Описание

Этот бот умеет модерировать, Повторять, Отображать аватары, Делать опции

Ссылка

.

Команды

  • Команда $help

    • Показывает список комманд

  • Команда $say

    • Повторяет сообщение

  • Команда $sayembed

    • Повторяет сообщение в коробке

  • Команда $предыстория

    • Показывает предысторию

  • Команда $invite

    • Выводит ссылку

  • Команда $kick

    • Кикает кого-либо

  • Команда $ban

    • Банит кого-либо

  • Команда $changelog

    • показывает изменение версий

  • Команда $poll

    • Делает опрос с опциями

  • Команда $avatar

    • Просматривает аватар юзера

  • Команда $play

    • Проигрывает аудио

  • Команда $stop

    • Останавливает аудио
.

Синтаксис

() - обязательно, [] - необязательно, {} - альтернативное написание

$help

$say (сообщение)

$sayembed (название) (описание)

Подсказка: если ввести например несколько слов в кавычки после $sayembed, то в названии будет несколько слов

$invite

$kick (Пользователь) [Причина]

$ban (Пользователь) [Причина]

$changelog [Версия]

$poll (Название) (Первая опция) (Вторая опция) [Опции вплоть до 9]

$avatar [Пользователь]

$play (Ссылка или файл)

$list

$stop

$rand {рандом, random} [первое число] [второе число]
